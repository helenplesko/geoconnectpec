Data Pre-Processing

All data was clipped to Prince Edward County 2021 census boundary, reprojected, filtered based on clients' needs. Fields were optimized by renaming them, adding aliases, and removing ones that weren't necessary for the project or future use.

Database Development

Two geodatabases were developed in ArcGIS Pro:

  1. One with a local projection of NAD 1983 UTM Zone 18N to be used internally.
  2. One with projection WGS 1984 Web Mercator (auxiliary sphere) for web purposes.

In both geodatabases:

  • Feature dataset for broadband data
  • Feature dataset for transit data
  • Domains for social infrastructure
  • Domains for road network
  • Subtypes for social infrastructure
