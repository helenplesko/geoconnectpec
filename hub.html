Solution: ArcGIS Hub

After collecting data from various sources that align with the vital signs theme, creating feature layers, web maps, dashbaords, experience builder we have created ArcGIS Hub as a platform to share the results with public. We have also added about page to provide information about the hub, how it was created and contact page which provides information about the Vital Signs of Prince Edward County.

This website is submitted in partial fulfillment of the requirements of the Geographic Information Systems Specialist program at Sir Sandford Fleming College.
