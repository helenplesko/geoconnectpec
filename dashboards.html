Solution: ArcGIS Dashboards

Public Transit

This dashboard is one stop interactive place for information about public transit in Prince Edward County. This dashboard includes a web map which has dizitized bus stops and routes overlain over population density map to provide context about the connectivity. This map includes the fare and schedule table as well as shows the number of fixed and on-demand bus stops within the map extent.

Household Type

This interactive dashboard includes two web maps which are One Person Per Household Type and Two or more Person Per Household Type. The indicators trigger the number of people per household in each dissemination areas.

Household Dwellings 2016 and 2021

This interactive dashboard include a web map which describes the Household Dwellings and the change occured during those years. The indicators trigger the change in number of household dwellings from 2016 to 2021.

Population Density

An interactive Dashboard was created which includes three web maps population density 2016, population density 2021 and the change in population density from 2016 to 2021. Indicators are used can be triggered with the map extent and the bar chart is also added which shows the change in density for each dissemination area.

Social Infrastructure

This interactive dashboard shows the locations of social infrastructure (e.g., community centers, libraries, town halls, etc.) within Prince Edward County, along with the fixed and on-demand transit routes. The pie chart can be used to filter by facility type. The list and pie chart are also triggered by map extent. Click on the features to learn more about them.

Household Education and Average Income 2021

This interactive dashboard allows users to compare the household attainment within each dissemination area, as well as the average household income. The indicator, lists, and pie charts are triggered by the map extents. Interact with the dashboard by clicking on the map surrounds and dissemination areas to learn more about the education levels and income of the population within them.
